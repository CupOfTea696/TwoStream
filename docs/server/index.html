Running TwoStream

  • Server
  • Options
  • Examples
  • Stopping the server

Server

Once you have installed and configured TwoStream, starting the WebSocket server is really easy. Just run the Artisan command below. To make sure the TwoStream server keeps running, you may use a process monitor such as Supervisor.

$ php artisan twostream:listen

Options

The twostream:listen command allows you to set a bunch of options when you start the Server.

  • -p or --port: Equivalent of websocket.port in the configuration.
  • -z or --push: Equivalent of push.enabled in the configuration. Set to true or false.
  • -Z or --push-port: Equivalent of push.port in the configuration.
  • -f or --flash: Equivalent of flash.allowed in the configuration. Set to true or false.
  • -F or --flash-port: Equivalent of push.port in the configuration.

Examples

# Run TwoStream on port 5555:
$ php artisan twostream:listen -p 5555

# Enable Push on port 2222:
$ php artisan twostream:listen -z true -Z 2222

Stopping the server

You can also stop the server by running the Artisan command below. If you are using Supervisor and want the server to automatically restart even when you manually stop it, you will need to set the autorestart option to true instead of the default unexpected.

$ php artisan twostream:stop
